Validador de Mayoría de Edad

Este es un pequeño script de JavaScript que interactúa con el usuario para determinar si es mayor o menor de edad basándose en su año de nacimiento.

Funcionalidades:

Validación de entrada: El programa no acepta campos vacíos ni texto en lugar de números.

Control de rango: Verifica que el año sea coherente (posterior a 1900 y no mayor al año actual).

Cálculo dinámico: Utiliza el objeto Date() de JavaScript para obtener el año real del sistema, evitando tener que actualizar el código manualmente cada año.

Opción de salida: Si el usuario presiona "Cancelar", el bucle se detiene correctamente.

Tecnologías utilizadas
JavaScript
Prompt/Alert API para la interacción básica.

Cómo probarlo
Abre la consola de tu navegador o vincula el archivo a un index.html.

Ingresa tu año de nacimiento cuando se te solicite.

El programa te dirá cuántos años tienes y tu estatus de mayoría de edad.
