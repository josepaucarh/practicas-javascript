Ejercicio 01: Cajero Automático

Este proyecto es una simulación de un cajero automático realizada en JavaScript puro.

Consigna:
Desarrollar un sistema que:
1. Valide un PIN de acceso (3 intentos).
2. Permita depositar dinero.
3. Permita retirar dinero (validando saldo suficiente).
4. Lleve un conteo de las operaciones realizadas.
5. Permita consultar el saldo actual.

Tecnologías usadas:
* JavaScript (Lógica, Bucles, Condicionales)
* HTML5
